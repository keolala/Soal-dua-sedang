Program Menghitung Volume Tabung – SoalDuaSedang
Deskripsi Program

Program ini dibuat menggunakan bahasa pemrograman Java.
Tujuannya adalah untuk menghitung volume tabung (silinder) berdasarkan jari-jari dan tinggi yang dimasukkan oleh pengguna.

Rumus yang digunakan:
Volume=π×r
2
×t

Keterangan:

r = jari-jari tabung

t = tinggi tabung

π = konstanta (3.14159265…)

Cara Kerja Program

Program meminta pengguna untuk memasukkan jari-jari tabung.

Program meminta pengguna untuk memasukkan tinggi tabung.

Nilai input tersebut dikirim ke fungsi hitungVolume(), yang akan menghitung volume dengan rumus matematika di atas.

Hasil perhitungan ditampilkan ke layar dalam format yang mudah dibaca.
